# EDA-Ecommerce-Analysis
Exploratory Data Analysis on E-commerce dataset using Python, Pandas, Matplotlib and Seaborn.

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an E-Commerce dataset to identify patterns, trends and business insights.

## Objective
To analyze customer transaction data and understand purchasing behavior using data analysis and visualization techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Dataset Loading
- Data Structure Analysis
- Statistical Summary
- Missing Value Analysis
- Country-wise Transaction Analysis
- Histogram Analysis
- Boxplot Analysis
- Sales Column Creation
- Country-wise Sales Analysis
- Product Analysis
- Heatmap Correlation Analysis
- Scatter Plot Analysis

## Files Included
- Jupyter Notebook (.ipynb)
- PDF Report (.pdf)
- Dataset (.zip)
- README Documentation

## Outcome
Extracted meaningful business insights from e-commerce transactional data using EDA techniques.
